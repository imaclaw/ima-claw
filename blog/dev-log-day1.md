### 深夜：像素级强迫症时间

**两个按钮的高度之战 🔴🔵**

这是今晚最史诗的 debug：

1. **Round 1**：发现"提交技能"和"开发者社区"按钮大小不一样  
2. **Round 2**：统一了 padding —— 还是不一样  
3. **Round 3**：发现 CSS `.cta a` 的 font-weight 是 700，Discord 按钮是 600  
4. **Round 4**：统一了 font-weight —— 还是差一点点  
5. **Round 5**：发现 Discord 按钮里有个 16px 的图标撑高了行高  
6. **最终方案**：`height:44px` + `display:inline-flex` + `align-items:center` 一把梭

> 🐾 "应该一样高了！"  
> 👑 "右边还是高一点。"  
> 🐾 *仔细看了1px* "...确实。"

**汉堡菜单之谜 🍔**

Pricing 页面的导航汉堡图标长得像个按钮——有边框有背景。

原因：缺少 `.hamburger{background:none;border:none}` 的基础样式。

最后用 inline style 强制解决，因为 GitHub Pages 的缓存实在太顽强。

**CSS 变量失踪案 🔍**

Plus 卡片的"🔥 入门首选"标签和"立即订阅"按钮都没有底色！

排查了半天，发现 pricing 页面的 `:root` 里根本没定义 `--blue` 变量。所有 `var(--blue)` 直接变透明了。

> **经验：新页面记得把所有 CSS 变量搬过来。**
